## 2019年7月15日
	调用已有自定义方法，连续作图，实现钻头的绘制。
	其中，钻头过渡阶梯和圆柱部分长度，需要分情况进行绘制。
	1. 圆柱长度分为：
		* 从钻头起点开始，即坐标原点开始；
		* 从钻尖末端开始，即钻尖末端坐标开始计算长度；
		* 圆柱台阶长度；
		* 圆柱台阶+过渡部分长度；

	2. 阶梯形状分为：
		* 锥形
		* 双锥
		* 2圆角
		* 3圆角
